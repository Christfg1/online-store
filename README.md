# Online Store

## Description
A Java command-line online store application that loads products from a CSV file. Users can view products, search inventory, add items to a cart, remove items, and checkout.

## Features
- Load products from Products.csv
- Display all products
- Search products by name
- Search products by price
- Search products by department
- Add products to cart
- Remove products from cart
- Display cart total
- Checkout with cash payment

## Technologies Used
- Java
- IntelliJ IDEA
- GitHub
- CSV file reading

## How to Run
1. Open the project in IntelliJ.
2. Make sure Products.csv is inside src/main/resources.
3. Run OnlineStoreApp.java.
4. Use the menu options in the console.
